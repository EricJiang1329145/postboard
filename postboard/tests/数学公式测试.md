# 数学公式测试文档

本文档包含多种经典数学公式，用于测试Markdown数学公式渲染功能。

## 1. 代数公式

### 1.1 二次方程求根公式

对于一般形式的二次方程 \( ax^2 + bx + c = 0 \)，其解为：

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

### 1.2 二项式定理

$$(a + b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k$$

其中 \( \binom{n}{k} = \frac{n!}{k!(n-k)!} \) 是二项式系数。

### 1.3 等差数列求和公式

$$S_n = \frac{n(a_1 + a_n)}{2} = na_1 + \frac{n(n-1)d}{2}$$

## 2. 几何公式

### 2.1 圆的面积和周长

- 面积：$A = \pi r^2$
- 周长：$C = 2\pi r$

### 2.2 三角形面积公式（海伦公式）

对于边长为 $a, b, c$ 的三角形，面积为：

$$S = \sqrt{s(s-a)(s-b)(s-c)}$$

其中 $s = \frac{a+b+c}{2}$ 是半周长。

### 2.3 勾股定理

在直角三角形中，斜边的平方等于两直角边的平方和：

$$a^2 + b^2 = c^2$$

## 3. 微积分公式

### 3.1 导数定义

函数 $f(x)$ 在点 $x_0$ 处的导数定义为：

$$f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$$

### 3.2 基本导数公式

- 幂函数：$\frac{d}{dx}(x^n) = nx^{n-1}$
- 指数函数：$\frac{d}{dx}(e^x) = e^x$
- 对数函数：$\frac{d}{dx}(\ln x) = \frac{1}{x}$

### 3.3 牛顿-莱布尼茨公式

$$\int_{a}^{b} f(x) dx = F(b) - F(a)$$

其中 $F(x)$ 是 $f(x)$ 的原函数。

## 4. 线性代数

### 4.1 矩阵乘法

对于矩阵 $A = (a_{ij})_{m\times n}$ 和 $B = (b_{ij})_{n\times p}$，它们的乘积 $C = AB$ 定义为：

$$c_{ij} = \sum_{k=1}^{n} a_{ik} b_{kj}$$

### 4.2 行列式

2×2矩阵的行列式：
$$\begin{vmatrix} a & b \\ c & d \end{vmatrix} = ad - bc$$

3×3矩阵的行列式：
$$\begin{vmatrix} a & b & c \\ d & e & f \\ g & h & i \end{vmatrix} = a(ei - fh) - b(di - fg) + c(dh - eg)$$

### 4.3 特征值和特征向量

若 $A$ 是 $n$ 阶方阵，$\lambda$ 是一个数，$x$ 是非零向量，且满足：

$$Ax = \lambda x$$

则 $\lambda$ 是 $A$ 的一个特征值，$x$ 是对应的特征向量。

## 5. 概率统计

### 5.1 期望和方差

- 期望：$E(X) = \sum_{i} x_i P(X = x_i)$（离散型）
- 方差：$Var(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2$

### 5.2 正态分布概率密度函数

$$f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x - \mu)^2}{2\sigma^2}}$$

其中 $\mu$ 是均值，$\sigma$ 是标准差。

### 5.3 大数定律

设 $X_1, X_2, \ldots, X_n$ 是独立同分布的随机变量，期望为 $\mu$，则对于任意 $\epsilon > 0$：

$$\lim_{n \to \infty} P\left(\left| \frac{1}{n} \sum_{i=1}^{n} X_i - \mu \right| < \epsilon\right) = 1$$

## 6. 物理学公式

### 6.1 牛顿第二定律

$$F = ma$$

### 6.2 爱因斯坦质能方程

$$E = mc^2$$

### 6.3 万有引力定律

$$F = G \frac{m_1 m_2}{r^2}$$

其中 $G$ 是万有引力常数，$m_1, m_2$ 是两个物体的质量，$r$ 是它们之间的距离。

## 7. 其他经典公式

### 7.1 欧拉公式

$$e^{i\theta} = \cos\theta + i\sin\theta$$

当 $\theta = \pi$ 时，得到欧拉恒等式：

$$e^{i\pi} + 1 = 0$$

### 7.2 黄金分割比

黄金分割比 $\phi$ 满足：

$$\phi = \frac{1 + \sqrt{5}}{2} \approx 1.618$$

### 7.3 黎曼ζ函数

$$\zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s}$$

## 8. 复杂公式示例

### 8.1 傅里叶变换

连续傅里叶变换：

$$F(\omega) = \int_{-\infty}^{\infty} f(t) e^{-i\omega t} dt$$

逆变换：

$$f(t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} F(\omega) e^{i\omega t} d\omega$$

### 8.2 拉格朗日方程

对于保守系统，拉格朗日方程为：

$$\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_i} \right) - \frac{\partial L}{\partial q_i} = 0$$

其中 $L = T - V$ 是拉格朗日量，$T$ 是动能，$V$ 是势能。
